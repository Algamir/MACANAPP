# MACANA

## Description
**MACANA** est une application mobile destinée à aider les amateurs de plantes à en prendre soin. Elle combine technologie et simplicité pour offrir une expérience enrichissante. Grâce à un système de capteurs connectés, MACANA permet de surveiller l'état de vos plantes et de recevoir des notifications lorsque vos plantes ont besoin d'attention.

L'application s'adresse à toutes les personnes souhaitant entretenir leurs plantes de manière plus efficace et connectée.

## Fonctionnalités principales
- Identification des plantes via l'API PlantNet.
- Notifications lorsqu'un capteur détecte que le sol est trop sec.
- Suivi de l'humidité du sol et de l'air grâce à un système connecté.
- Boîtier imprimé en 3D pour le matériel.

## Lacement
Pour lancer l'application, exécutez simplement :

npx expo start
